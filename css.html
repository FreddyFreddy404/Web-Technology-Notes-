1. Basics, Priority & Selectors

3 Ways to Add CSS (Priority Order)

Priority dictates what happens if rules conflict. Inline > Internal > External.

<!-- 1. INLINE (Highest Priority) -->
<h1 style="color: red;">Title</h1>

<!-- 2. INTERNAL (<style> tag in <head>) -->
<style>
  h1 { color: green; }
</style>

<!-- 3. EXTERNAL (Best Practice) -->
<link rel="stylesheet" href="style.css">

rel="stylesheet": Tells browser the relationship of the linked file is a stylesheet.

href: Hypertext Reference (path to file).

style.css: Industry standard name for main stylesheet, just like index.html.

Selectors Exhaustive Breakdown

/* 1. Element Selector */
h1 { color: blue; }

/* 2. Universal Selector (Everything) */
* { margin: 0; }

/* 3. Class Selector (Multiple use) */
.card { background: yellow; }

/* 4. ID Selector (Unique, once per page) */
#hero { font-size: 30px; }

/* 5. Grouping Selectors */
h1, h2, h3 { font-family: Arial; }
h1 { color: blue; }

Heading 1

Background Image

Cover vs Contain are crucial for fitting images into divs.

background-size: cover;

Stretches image to completely cover box. Edges might be cropped.

background-size: contain;

Forces entire image to be visible. Might leave empty space.

Exhaustive Positioning

position: static;

Default. Flows normally. top/left do nothing.

position: relative;

Moves relative to its *normal* starting point.

position: absolute;

Removed from flow. Relies on nearest *positioned parent*.

position: fixed;

Sticks to the screen/browser window. Never moves on scroll.

position: sticky;

Like you said: Constrained to its parent. Scrolls normally until it hits the top, then sticks until the parent div ends.

Units & Parents/Children

Parent/Child: A div (Parent) wrapping a p (Child). Absolute elements position themselves relative to their Parents!

Units:
px: Absolute pixels.
em: Relative to Parent font size.
rem: Relative to Root (HTML) font size.
vh / vw: 100vh is 100% of Screen Height.

6. Media Queries

Responsive Design Exhaustive

/* Exact Match */
@media (width: 600px) { ... }

/* "At 600px AND AFTER (Larger / PC)" */
@media (min-width: 600px) { ... }

/* "At 600px AND BEFORE (Smaller / Mobile)" */
@media (max-width: 600px) { ... }

/* Range: Between 200px and 300px */
@media (min-width: 200px) and (max-width: 300px) { ... }
